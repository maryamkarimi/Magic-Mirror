How to build and run our submission.

Step 1 - Install needed libraries.
- Run "sudo apt-get update"
- Run "sudo apt-get install libcurl4-openssl-dev"
- Run "sudo apt-get install libsdl1.2-dev libsdl-image1.2-dev libsdl-mixer1.2-dev libsdl-ttf2.0-dev"

Step 2 - Run the command "make" in the folder with all the .cpp and .h files.

Step 3 - Run ./Test to run the application.


** Reminders are saved in Reminders directory, to create your reminder file,
make a file called yourusername_reminders.txt and sign in with that username
